# TechFlow

**Enterprise Ticket Management System**

**TechFlow** is a robust technical ticket management solution designed to optimize communication between business departments and IT teams. The system enables ticket creation, triage, and resolution while ensuring traceability, organization, and operational efficiency.

## Overview

In enterprise environments, the lack of a clear internal support workflow leads to delays, rework, and loss of context. TechFlow was created to centralize this process in a single platform, with clear business rules and a complete history for every ticket.

## Key Features

- **Hierarchy Management:** organizational structure based on **Departments** and **Employees**.
- **Ticket Lifecycle:** full control over statuses (**Open**, **In Progress**, **Resolved**) and priority levels.
- **Resolution History:** detailed records of troubleshooting steps and solutions linked to each ticket.
- **Full Traceability:** end-to-end tracking of each request, from creation to closure.
- **Responsive Interface:** built with **Django Template Language (DTL)** and **Tailwind CSS**, following **Logical Brutalism** principles.

## Tech Stack

- **Backend:** Python 3.12, Django (ORM, Auth, Templates)
- **Frontend:** DTL, Tailwind CSS, Alpine.js
- **Database:** PostgreSQL (production) or SQLite (development)
- **Tooling:** dependency management and execution with **uv**

## Project Goals

- Improve communication between requesting teams and IT
- Reduce average response and resolution times
- Standardize the technical support workflow
- Ensure transparency and an auditable ticket history
